Bboss

bboss is a j2ee framework include aop/ioc,mvc,persistent,taglib,rpc,event ,bean-xml serializable ，session sharing and so on.

Download this project as a .zip file Download this project as a tar.gz file

bboss特色： 

1.aop/ioc业务组件管理、依赖注入（属性注入、构造器注入、工厂模式注入）、声明式事务等   

2.rpc(http/netty/mina/jms/webservice/rmi/jgroups/restful) 安全高效 

3.distribute event framework 

4.jsp taglib 分页、树、列表、抽屉式、逻辑标签 

5.mvc/restful（很好地支持jsonp，传统/html5文件上传下载，json）

6.国际化组件（属性文件热加载、缓冲、高性能、支持从classpath和web应用目录中加载资源文件） 

7.persistent framework，全局事务管理（可整合托管bboss，ibatis，hibernate，spring等持久层的事务），很好地和业界主流数据源结合（dbcp,c3p0,proxool,weblogic,druid等） 

8.xml-bean serializable（高效，很好地支持各种java数据类型和复杂对象结果，支持引用关系的序列化）

9.jms 开发套件 

10.cxf webservice 服务发布和客户端代理 

11.cluster with jgroups 

12.quartz任务引擎管理 

13.hession管理及发布，客户端调用支持 

14.集群会话共享，跨容器跨平台，跨站跨应用会话共享及SSO，高效，配置简单 

15.令牌管理（令牌生成和校验），支持集群环境令牌管理和校验 

16.完备的框架监控机制
